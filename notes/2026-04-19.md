### 직원 상세 페이지 구조
```
헤더 (이름/직책/시급)          [정보 수정] [월 필터]
급여 요약 카드 4개 (근무시간/세전/세금/실수령액)
```

### 주요 기능
- 출퇴근 달력: 날짜 클릭 → 하단 인라인 폼으로 출근/퇴근 시간 입력
- 각 날짜 셀에 근무시간(h) + 시간대 표시
- 일요일(빨강)/토요일(파랑) 색상 구분
- 오늘 날짜 강조 (다크그린 원)
- 근무 기록 삭제 버튼 (✕)
- 급여 자동계산: 요일별 시급 × 시간 + 주휴수당(주 15h 이상 시)
- 과거 인건비: 근무 기록 없이 연/월/금액 직접 입력 가능

## 2. 버그 수정

### 직원 추가 500 에러
- **원인**: Supabase `staff` 테이블에 `sunday_hourly_pay`, `tax_rate` 컬럼 없음
- **해결**: Supabase SQL Editor에서 ALTER TABLE 실행
  ```sql
  ALTER TABLE staff 
  ADD COLUMN IF NOT EXISTS sunday_hourly_pay integer,
  ADD COLUMN IF NOT EXISTS tax_rate numeric(5,2) DEFAULT 3.3;
  ```
- **추가**: `work_logs` 테이블도 새로 생성 (출퇴근 기록용)

### 근무시간 자동 계산
- **원인**: `work_logs` 테이블에 `hours_worked` 컬럼이 있으나, API에서 계산 없이 insert
- **해결**: `app/api/work-logs/route.ts`에서 출퇴근 시간 차이 계산 후 저장
  ```typescript
  function calcHours(start: string, end: string): number {
    // 분 단위 차이 → 시간 단위 변환 (자정 넘어가는 경우 처리)
  }
  ```

### 대시보드 지출 0 표시
- **원인**: `profit`이 generated column인데 UPDATE 시 profit까지 업데이트하려 해서 오류
- **해결**: `total_expense`만 업데이트 (profit은 자동 재계산됨)
  ```sql
  UPDATE monthly_summary 
  SET total_expense = (
    SELECT COALESCE(SUM(amount), 0)
    FROM monthly_expenses
    WHERE monthly_expenses.year = monthly_summary.year
      AND monthly_expenses.month = monthly_summary.month
  );
  ```

### 엑셀 파싱 금액 불일치
- **원인 1**: 숫자에 콤마 포함 → `Number("1,234,567")` = NaN
- **원인 2**: Excel 날짜 시리얼 숫자가 그대로 읽힘
- **해결**: `cellDates: true` 옵션 + 콤마 제거 파서 추가
  ```typescript
  const workbook = XLSX.read(buffer, { type: 'array', cellDates: true })
  function parseAmount(val: unknown): number | null {
    const str = String(val).replace(/,/g, '').trim()
    return isNaN(Number(str)) ? null : Number(str)
  }
  ```

## 4. Supabase SQL 요약 (실행 완료)

```sql
-- 1. staff 컬럼 추가
ALTER TABLE staff 
ADD COLUMN IF NOT EXISTS sunday_hourly_pay integer,
ADD COLUMN IF NOT EXISTS tax_rate numeric(5,2) DEFAULT 3.3;

-- 2. work_logs 테이블 생성
CREATE TABLE IF NOT EXISTS work_logs (
  id bigserial PRIMARY KEY,
  staff_id bigint REFERENCES staff(id) ON DELETE CASCADE,
  date date NOT NULL,
  start_time time NOT NULL,
  end_time time NOT NULL,
  hours_worked numeric(4,2) NOT NULL DEFAULT 0,
  created_at timestamptz DEFAULT now()
);

-- 3. 기존 지출 데이터 반영
UPDATE monthly_summary 
SET total_expense = (
  SELECT COALESCE(SUM(amount), 0)
  FROM monthly_expenses
  WHERE monthly_expenses.year = monthly_summary.year
    AND monthly_expenses.month = monthly_summary.month
);
```
